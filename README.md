About the Project

# Overview

This project uses a Convolutional Neural Network (CNN) to classify spine conditions from medical images. The model predicts three classes:

* Normal
* Scoliosis
* Spondylolysis

The aim is to support early detection using image-based analysis.

---

# Features

* Basic data analysis (EDA)
* Image preprocessing and augmentation
* CNN model for classification
* Accuracy and loss visualization
* Confusion matrix for evaluation
* Prediction on new images

---

# Dataset

The dataset is organized into three classes:

* Normal
* Scoliosis
* Spondylolysis

Each class contains corresponding images for training, validation, and testing.

# Model

The model is a simple CNN with:

* Convolution + MaxPooling layers
* Dense layer
* Softmax output (3 classes)
